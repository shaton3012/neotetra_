NEOTETRA v2.2.0
!
NEOTETRA は、80年代のCity Popとサイバーパンクな世界観を融合させた、次世代型テトリス・クローンです。Web Audio APIによるアルゴリズム演奏と、リッチなエフェクトが特徴です。
特徴
	•	City Pop Algorithm BGM: プレイ状況に合わせて生成されるメロウなシンセサウンド。
	•	Combo System: 連続ライン消去によるスコア倍増。
	•	Cyberpunk UI: ターミナルログやネオンエフェクトによる没入感のあるデザイン。
	•	System Repair: スコアを稼ぐことで「都市を復旧（クリア）」するゲーム性。
セットアップ
# リポジトリをクローン
git clone [https://github.com/あなたのユーザー名/neotetra-game.git](https://github.com/あなたのユーザー名/neotetra-game.git)

# ディレクトリへ移動
cd neotetra-game

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
技術スタック
	•	React
	•	Tailwind CSS
	•	Lucide React (Icons)
	•	Web Audio API (Synthesized BGM)
